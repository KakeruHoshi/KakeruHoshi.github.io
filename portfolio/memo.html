・パラメータについて

ユーザーが入力した情報を受け取るのがparams。
paramsの中身は↓こんな感じ
（ハッシュ）
params = {
　　"utf-8" => ✔︎,
　　"key" => "value",
　　"tweets" => {
　　　　"body" => "これがツイートの中身だお"
　　}
}



・View⇨controllerの値受け渡し

リンク

基本

「View」　
example_path(user_id: "1", company_id: "3")
↓
↓ Parameters {"user_id"=>"1", "company_id"=>"3"}
↓
「Controller」
params[:user_id]
params[:company_id]

変数渡し

「View」　
<%= link_to "リンク名", example_path(@ex) %>

「Controller」
def example
　　@ex = Ex.find_by(id: params[:format]) *.find_byはデータを指定して取り出す。
end


パラメータを一つだけ単体渡し

<%= hidden_field_tag :email, @user.email %>
